# Bagel
The implementation of 'Robust and Unsupervised KPI Anomaly Detection Based on Conditional Variational Autoencoder'
Models are in `model.py`.

# Dependencies
python >= 3.7

``` bash
pip install -r requirements.txt
```


# Run
``` bash
python main.py
```
